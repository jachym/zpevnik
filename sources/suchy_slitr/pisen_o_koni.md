### Píseň o koni

```
C                      F               
A proto odpouští má láska mému koni, 
G           F C G                    
na cestě od tebe tu jeho loudavost,
           C                  F                   
vždyť když tě opustím, nač spěchat, co nás honí,
      G                        C  F C     
dokud se nevracím, je na spěch času dost.
```
